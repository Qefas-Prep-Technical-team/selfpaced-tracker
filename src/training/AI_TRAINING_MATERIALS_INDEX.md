## 🎓 Key Facts to Remember

### **Platform Basics**

- **Roles:** Admin, Teacher, Student, Parent
- **Access:** Role-based, subscription-limited
- **Devices:** Web (full) + Mobile (app, growing parity)
- **Login:** Email/password, OAuth (Google) available

### **Main Features**

- Academic: Exams, grades, assignments, classes, timetable
- Behavioral: Attendance, conduct tracking, behavior alerts
- Communication: Notifications, parent-teacher messaging
- Billing: Subscription plans, payment processing (Paystack)
- School Customization: Landing pages, branding, settings

### **Key URLs**

- Login: `/login`
- Signup: `/signup`
- Verification: `/verification`
- Join school: `/join/school/[code]`
- Join class: `/join/class/[id]`
- School invite: `/pricing` or landing page

### **Common Paths**

- **Student Dashboard:** `/dashboard/student/`
- **Teacher Dashboard:** `/dashboard/teacher/`
- **Parent Dashboard:** `/dashboard/parent/`
- **Admin Dashboard:** `/dashboard/admin/`
- **Settings:** `/dashboard/[role]/settings`
- **Exams:** `/Exams&Quizzes` (main) or Dashboard → Exams

### **Subscription Limits**

Vary by tier, but typically controlled:

- maxStudents
- maxTeachers
- maxClasses
- maxExams
- maxStorageGb
- AI usage limits
- Custom domain access

### **File Upload Limits**

- Max per file: ~50MB
- Supported types: PDF, DOC, XLS, PNG, JPG, GIF, ZIP, MP4, etc.
- Storage limit: Based on subscription tier

### **Payment Details**

- Gateway: Paystack
- Currencies: GHS (Ghana), NGN (Nigeria)
- Billing: Monthly or yearly
- Trial: Configurable days (typically 14)

### **Browser Support**

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile Safari (iOS 12+)
- Chrome Android

## 📱 Web vs Mobile Quick Reference

### **Features Available on Both (Mostly Parity):**

✅ Dashboard / Home  
✅ Grades viewing  
✅ Exams - taking (web fully, mobile fully)  
✅ Assignments - viewing & submitting  
✅ Classes - viewing enrolled classes  
✅ Timetable - viewing schedule  
✅ Attendance - viewing only  
✅ Profile - basic info  
✅ Notifications - push + in-app  
✅ Login/authentication

### **Web Only (Not on Mobile):**

❌ Creating exams  
❌ Creating classes  
❌ Creating assignments  
❌ Creating timetable  
❌ Admin settings  
❌ Landing page builder  
❌ School management  
❌ User management  
❌ Analytics/reports (limited on mobile)  
❌ AI features (mostly)  
❌ Payment/billing management

### **Mobile Strengths:**

✅ Easier attendance marking (one-handed)  
✅ Push notifications (more reliable)  
✅ Swipe navigation (exam questions)  
✅ Bottom tab navigation (thumb-friendly)  
✅ Offline capability (planned)
