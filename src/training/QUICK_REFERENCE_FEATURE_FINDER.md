# SchoolHub - Quick Reference Feature Finder

**For AI assistants to quickly look up where features are located and how to access them**

---

## 🔍 Quick Lookup by Question

### "Where do I find...?"

#### **GRADES & ACADEMIC RECORDS**

**"Where can I check my grades?"**
- **Web (Student):** Dashboard → Grades tab
- **Web (Teacher):** Dashboard → Grades
- **Web (Parent):** Dashboard → Child Performance → Grades for [Child Name]
- **Mobile (Student):** Tap "Grades" in bottom navigation
- **Mobile (Teacher):** Grades tab in bottom navigation

**"Where can I see my exam scores?"**
- **Web:** Dashboard → Exams → Click exam to view results
- **Mobile:** Exams tab → Select exam → View Results

**"Where do I enter/upload grades?"**
- **Web (Teacher/Admin):** Dashboard → Grades → "+ New Grade"
- **Web (Teacher/Admin Alternative):** Dashboard → Assignments → [Assignment] → Grade Submissions

**"Where can I download my grade report?"**
- **Web:** Dashboard → Grades → Download button (top right)
- **Mobile:** Not available on mobile, use web version

---

#### **ASSIGNMENTS & HOMEWORK**

**"Where do I find assignments?"**
- **Web (Student):** Dashboard → Assignments
- **Mobile (Student):** Tap "Assignments" tab or scroll in Dashboard
- **Web (Teacher):** Dashboard → Assignments

**"How do I submit an assignment?"**
- **Web:** Dashboard → Assignments → [Assignment] → Click "Submit" → Upload file → Confirm
- **Mobile:** Assignments tab → [Assignment] → "Submit" → Select file → "Upload"

**"Where can I see assignment feedback?"**
- **Web (Student):** Dashboard → Assignments → [Assignment] → View Feedback section
- **Mobile (Student):** Assignments tab → [Assignment] → Scroll to see teacher's feedback

**"Where do I create an assignment?"** *(Teacher/Admin only)*
- **Web:** Dashboard → Assignments → "+ New Assignment"
- Choose class(es)
- Add title, instructions, due date
- Attach files if needed
- Set rubric/points
- Click "Create"

**"Where can I see who hasn't submitted?"**
- **Web (Teacher):** Dashboard → Assignments → [Assignment] → View "Submission Status"
- Shows pending/submitted/late

---

#### **EXAMS & ASSESSMENTS**

**"Where do I take an exam?"**
- **Web:** Dashboard → Exams → [Exam Name] → "Start Exam" button (if available)
- **Mobile:** Exams tab → [Exam Name] → "Start" button
- **Timer** displays at top during exam

**"Where do I create an exam?"** *(Teacher/Admin only)*
- **Web:** 
  - Top menu → Click "Exams & Quizzes" 
  - OR Dashboard → Settings → Exams & Quizzes
  - Click "+ New Exam"
  - Add title, select classes, set date/time
  - Click "Add Questions"
  - Add each question with type (multiple choice/true-false/short answer/file upload)
  - Click "Publish" when ready

**"Where do I see exam results?"**
- **Web (Student/Parent):** Dashboard → Exams → [Exam] → View grade/score once released
- **Web (Teacher):** Dashboard → Exams → [Exam] → View all student attempts
- **Mobile (Student):** Exams tab → Completed exams section

**"Where can I use AI to generate exam questions?"** *(Admin/Teacher, paid plan only)*
- **Web:** Exams & Quizzes → "+ New Exam" → While adding questions → "AI Generate" button
- Type topic/subject
- Select number of questions
- Choose question type
- Click "Generate"
- AI creates questions, you can edit/approve

#### **ATTENDANCE**

**"Where can I check attendance?"**
- **Web (Student/Parent):** Dashboard → Attendance Report
- **Mobile (Student):** Not available, use web
- **Web (Admin):** Dashboard → Settings → Attendance Reports

**"Where do I mark attendance?"** *(Teacher/Admin only)*
- **Web:** Dashboard → Attendance → Select class → Select date → Check/uncheck students → Save
- **Mobile (Teacher):** Attendance tab → Select class → Check boxes for present students → Save

**"Where can I download attendance records?"**
- **Web:** Dashboard → Attendance → Download button
- **Mobile:** Not available, use web

#### **CLASSES & ENROLLMENT**

**"Where do I see my classes?"**
- **Web (Student/Teacher):** Dashboard → Classes
- **Mobile (Student/Teacher):** Classes tab in bottom navigation

**"Where do I enroll in a new class?"** *(Admin/Teacher)*
- **Web:** Dashboard → Classes → [Class name] → "+ Enroll Student" → Search student → Add
- **Invite option:** Share invite code with student

**"Where can I create a new class?"** *(Teacher/Admin)*
- **Web:** Dashboard → Classes → "+ New Class"
- Enter class name
- Select grade/level
- Select subjects
- Click "Create"

**"Where can I see class roster?"**
- **Web (Teacher/Admin):** Dashboard → Classes → [Class] → Student List
- **Web (Student):** Dashboard → Classes → [Class] → View Classmates
- **Mobile:** Classes tab → [Class] → Roster

#### **TIMETABLE/SCHEDULE**

**"Where do I check the timetable?"**
- **Web:** Dashboard → Timetable
- **Mobile:** Timetable tab (bottom navigation)
- **View:** Day, Week, or Month view

**"Where can I create a timetable?"** *(Admin only)*
- **Web:** Dashboard → Settings → Timetable → "+ Create Timetable"
- Select class
- Add time periods (period 1, period 2, etc.)
- Assign subjects and rooms
- AI auto-conflict check
- Click "Generate"

#### **SCHOOL SETTINGS & ADMINISTRATION**

**"Where do I manage school information?"** *(Admin)*
- **Web:** Dashboard → Settings
- Update:
  - School name, logo, colors
  - Email preferences
  - User management (view/edit teachers, students, parents)
  - Academic sessions and terms
  - Payment settings
  - Maintenance mode toggle

**"Where do I create a custom landing page?"** *(Admin)*
- **Web:** Dashboard → "Subdomain" or "Settings" → "Create Landing Page"
- Enter subdomain (e.g., "myschool")
- Click "Edit Landing Page"
- **In the builder:**
  - Add hero image
  - Add school description
  - Add features/highlights
  - Add testimonials section
  - Add photo gallery
  - Customize colors
  - Set accent color
  - Toggle dark mode
  - Click "Save"
- **Your site:** myschool.schoolhub.com

**"Where can I manage users?"** *(Admin)*
- **Web:** Dashboard → Settings → Users
- View list of all users
- Click user to edit roles/permissions
- Click "+" to invite new user
- Send invite → User registers

**"Where do I manage academic sessions?"** *(Admin)*
- **Web:** Dashboard → Settings → Academic Sessions
- Click "+ New Session"
- Enter session name (e.g., "2025/2026")
- Set start/end dates
- Define terms (1st, 2nd, 3rd)
- Click "Create"

**"Where do I manage subjects?"** *(Admin)*
- **Web:** Dashboard → Settings → Subjects
- Click "+ New Subject"
- Enter subject name (Math, English, etc.)
- Assign to departments
- Select classes it applies to
- Click "Create"

- Shows:

#### **BEHAVIORAL TRACKING**

**"Where do I check conduct/behavior score?"**
- **Web (Student):** Dashboard → Profile → Conduct Score section
- **Web (Parent):** Dashboard → Child Performance → Behavior section
- **Mobile:** Profile tab → Conduct section

**"Where do I report a behavior incident?"** *(Teacher/Admin)*
- **Web:** Dashboard → "Behavior Alerts" or "Disciplinary" → "+ New Alert"
- Select student
- Describe incident
- Categorize (bullying, academic dishonesty, etc.)
- Click "Report"
- Parent notified if severity high

**"Where can I see student history/timeline?"** *(Admin)*
- **Web:** Dashboard → Student Details → Student History tab
- Shows all events:
  - Enrollment dates
  - Grade changes
  - Behavior incidents
  - Promotions/transfers
  - Achievements

#### **PAYMENTS & BILLING**

**"Where can I view my subscription?"**
- **Web (Admin/User):** Dashboard → Settings → Subscription/Billing
- Shows:
  - Current plan name
  - Renewal date
  - Monthly/yearly status
  - Limits (students, classes, storage, exams)
  - Usage vs limits

**"Where do I upgrade my plan?"**
- **Web:** Dashboard → Settings → Subscription → "Upgrade Plan" button
- Select new tier
- Review pricing
- Proceed to Paystack payment
- Confirm with card/mobile money
- Plan upgrades immediately

**"Where can I view past payments/invoices?"**
- **Web (Admin):** Dashboard → Settings → Payments → Invoice History
- View all past invoices
- Filter by date
- Download as PDF
- Resend invoice to email

**"How do I change billing cycle (monthly to yearly)?"**
- **Web:** Dashboard → Settings → Billing → "Change Billing" option
- Select yearly (usually 20% discount)
- Confirm
- Next charge is yearly amount

**"Where are transaction records?"**
- **Web (Admin):** Dashboard → Settings → Payments → Transactions
- View all transactions
- Date, amount, status, payment method
- Download report

#### **NOTIFICATIONS & COMMUNICATIONS**

**"Where do I see notifications?"**
- **Web:** Bell icon (🔔) in top right corner
- **Mobile:** Notifications tab or section in menu
- Shows: Bell (in-app) + Email + Push notifications

**"How do I enable/disable notifications?"**
- **Web (All roles):** Dashboard → Profile → Notification Settings
- Toggle on/off for:
  - Grade updates
  - Assignment alerts
  - Attendance alerts
  - Exam announcements
  - Parent messages
  - System notifications

**"Where can I link to a parent?"** *(Student)*
- **Web:** Dashboard → Profile → "Link Parent" button
- Share unique link/code with parent
- Parent clicks link or enters code
- You approve link request
- Parent can now see your grades, attendance

**"Where can I link to my child?"** *(Parent)*
- **Web:** Dashboard → "Link Child" section
- Enter child's student ID or email
- Send request
- Child or admin approves
- Once approved, you see their:
  - Grades
  - Attendance
  - Performance
  - Announcements

---

#### **PROFILE & ACCOUNT**

**"Where do I change my password?"**
- **Web:** Dashboard → Profile → "Change Password" or Settings
- Enter old password
- Enter new password (min 8 chars, mixed case/numbers/symbols)
- Confirm
- Click "Update"

**"Where do I change my profile picture?"**
- **Web:** Dashboard → Profile → Click avatar → "Upload Photo"
- Select image from device
- Click "Upload"

**"Where do I verify my email?"** *(If not verified)*
- Check email for verification link
- Click link in email
- OR go to `/verification` page
- Enter code from email
- Click "Verify"

**"Where do I see my account information?"**
- **Web:** Dashboard → Profile
- Shows:
  - Name, email
  - Phone (if added)
  - School enrollment
  - User role
  - Status (Active/Inactive)

## 🚀 Most Common User Flows

### **Student: First Time Setup**
1. Signup at `/signup` (or use join code `/join/school/[code]`)
2. Verify email
3. Complete onboarding
4. Login to dashboard
5. Explore Classes
6. Check Timetable
7. View Exams (take if available)

### **Teacher: Start of Year**
1. Login to dashboard
2. Go to Classes → Create Class
3. Add students to class (enroll or share invite code)
4. Go to Settings → Academic Sessions → Create session if not done
5. Create Exams in Exams & Quizzes
6. Create Assignments
7. Create Timetable in Settings
8. Start marking attendance
9. Create grades

### **Admin: School Setup**
1. Configure School Info (Settings)
2. Create Academic Session (Settings)
3. Create Terms (Settings)
4. Create Classes (Classes)
5. Create Subjects (Settings)
6. Invite Teachers
7. Invite Students (or share signup link)
8. Optionally create custom landing page (Subdomain)
9. Configure subscription (Settings → Billing)

### **Parent: Monitor Child**
1. Login to dashboard
2. Link Child (if not auto-linked)
3. Wait for approval
4. View child's grades (Child Grades)
5. View attendance (Child Attendance)
6. Check announcements
7. Enable notifications
8. Message teacher if needed

## 🔧 Troubleshooting Quick Links

**"I can't find [feature]"**
→ Check if you have the right role (see Role permissions above)
→ Check subscription tier (might need upgrade)
→ Try searching this document for exact feature name

**"I get 'Feature not available'"**
→ Your plan limit reached (Settings → Subscription → Upgrade)

**"I forgot password"**
→ Login page → "Forgot Password" link → Check email for reset link

**"Can't see student's grades"**
→ Check if student is enrolled in class
→ Check if grades have been released (not in draft)
→ Check if you have teacher role for that class

**"Assignment not submitting"**
→ Check file size (max ~50MB)
→ Check upload storage (might be full)
→ Try different browser

**"Exam won't start"**
→ Check if exam is published (not draft)
→ Check if you're a student in that class
→ Check if exam date/time has arrived

## 🎓 Teaching Recommendations

**For questions like "How do I...":**
1. Identify the user's role
2. Check their subscription tier
3. Look up feature in this guide
4. Provide exact navigation path
5. Mention if feature is only on web or also mobile
6. If they're hitting a limit, mention upgrade path

**For "Where is...?" questions:**
1. Use the Feature Availability by Role table
2. Give web path first (more complete)
3. Add mobile path if available
4. Include exact button names and menu items

**For signup/account issues:**
Direct to `/login`, `/signup`, `/verification`

**For data/grade/attendance questions:**
Ask their role first (affects what they can see)
